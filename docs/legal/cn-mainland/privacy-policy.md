本隐私政策适用于我们在中国大陆部署与运营的服务集群下向您提供的「虾传」及相关客户端（包括移动应用、桌面应用、鸿蒙应用等）与 **Web 网页端**（以下合称「本产品」或「服务」）。**本版本仅适用于您选择连接中国大陆服务节点（例如 `api.xiachuan.net` / `wss://ws.xiachuan.net`）的情形。**

## 1. 我们处理哪些信息

### 1.1 账号与身份验证

- **电子邮箱**：用于注册、登录、接收验证码与重要通知。
- **密码**：以加密摘要形式存储，我们无法还原您的明文密码。
- **昵称 / 显示名称**（若您提供）：用于在设备列表、会话界面等处的展示。
- **验证码**：用于注册或登录流程；仅在验证所需期限内短期保存。
- **手机号码**（如我们向您提供或您使用手机号验证功能且经您单独同意）：用于发送短信验证码；短信发送可能由**腾讯云短信**等服务商处理，详见「第三方服务」。

### 1.2 设备与终端信息

为在多终端间同步消息与文件，我们会处理：

- **设备标识**（例如应用生成的 `deviceId`）、**平台类型**（如 iOS、Android、Windows、macOS、Linux、Web、HarmonyOS）、**设备名称**（可由您修改）。
- **会话版本 / 登录态信息**：用于安全校验与多端管理。

### 1.3 服务使用与通信元数据

- **实时连接与订阅信息**：通过**消息中间件**维护的 WebSocket 连接、频道订阅关系（例如用户维度频道），用于投递消息与传输信令。
- **消息与文件的元数据**：例如消息类型、时间戳、与存储相关的键名或对象键、文件大小、分片上传任务标识等。**您通过「自有对象存储」传输的文件内容，由您配置的存储服务商网络直连上传/下载；我们不会将此类文件内容作为「托管网盘」长期保存在自有服务器上。** 若您未使用自有存储，相关技术路径以实际产品配置为准。
- **局域网 / 直连传输**：在「指定设备」等场景下，文件数据可能经您局域网或 WebRTC 等通道在设备间传输；该路径下内容通常**不经由我们的服务器持久化存储**，但为实现发现、握手与信令，仍可能产生连接与诊断类元数据。

### 1.4 网络与日志

为保障安全、排障与合规，服务器可能记录：

- **访问时间、接口路径、错误码、来源 IP** 等访问日志（在合理期限内保存，具体以内部制度为准）。
- **异常与安全事件**相关信息。

### 1.5 会员与交易（若您购买会员）

- **订单号、支付状态、会员档位、设备数上限、增购包数量**等履约所必需的信息。
- 通过**支付宝**或 **Apple 内购（经 RevenueCat 对接）** 完成支付时，支付机构会按其规则处理交易信息；我们仅接收与权益发放相关的**必要回执字段**。

### 1.6 产品改进与统计分析

我们可能通过**第三方统计分析服务**收集**去标识或聚合**的使用情况，例如应用版本、屏幕视图、部分操作的成败与**分桶后的统计维度**（例如文件体积分档、文本长度分档）。**我们不会将聊天正文、完整文件名、对端设备的明文标识符用于分析上报。**

## 2. 系统权限与本地能力说明

不同操作系统以「权限」或「隐私敏感能力」形式向您申请授权。以下说明与当前产品实现及典型用途一致；**若某版本未包含某项能力，以您设备上实际弹窗为准**。

### 2.1 Android（非 Google Play 分发包，典型权限）

| 权限或能力 | 用途说明 |
|------------|----------|
| **网络访问（INTERNET）** | 连接 API、实时通道、对象存储预签名地址等。 |
| **相机（CAMERA）** | 扫描二维码登录、传输相关场景。 |
| **存储 / 媒体读取与写入** | 选择文件、保存接收到的图片/视频到相册、缓存与断点续传等。 |
| **读取媒体图片/视频/音频** | 读取您主动选择的媒体以便发送。 |
| **Wi‑Fi 状态与变更、精确定位/大致位置、附近 Wi‑Fi 设备** | 用于 **Smart Link / Wi‑Fi Direct / 局域网发现与直连**；**并非用于持续地理位置追踪**。在支持「neverForLocation」的附近 Wi‑Fi 权限场景下，按系统要求不将权限用于推导精确位置。 |
| **列出已安装应用（QUERY_ALL_PACKAGES）** | 部分构建中的 **APK 扫描/发送/安装** 相关能力（若该版本提供）。 |
| **请求安装包（REQUEST_INSTALL_PACKAGES）** | 在征得您同意后安装 APK（若该版本提供）。 |

**Google Play 分发变体**：为符合应用商店政策，该变体可能**不包含**「列出全部应用」「请求安装未知应用」等权限；对应功能在 Play 版本中**不可用或已移除**。

文件夹选择与自定义保存路径通过**系统目录选择器（SAF）**完成；转存到系统「下载」目录在 Android 10+ 通过 **MediaStore** 实现。

### 2.2 Android（Google Play 分发变体）

在通过 Google Play 安装的版本中，**不包含** `QUERY_ALL_PACKAGES` 与 `REQUEST_INSTALL_PACKAGES` 权限；我们不会通过该版本请求上述权限。

### 2.3 iOS / iPadOS

| 用途说明（系统弹窗文案） | 能力 |
|--------------------------|------|
| **本地网络** | 用于发现并连接局域网内的设备以传输文件；涉及 Bonjour 服务（如 `_ultrasend._tcp`、`_wifi-direct._tcp` / `_wifi-direct._udp` 等，以系统展示为准）。 |
| **相机** | 用于扫描二维码登录。 |
| **相册（读取/写入）** | 用于将接收的图片、视频保存到相册；读取您选择的媒体以便发送。 |

若您使用 **iOS 分享扩展**，系统可能将您从其他应用分享的内容交由扩展处理；我们仅在为完成您发起的分享与发送所必需的范围内处理。

### 2.4 macOS

| 用途说明 | 能力 |
|----------|------|
| **本地网络** | 发现局域网设备并进行局域网文件传输（Bonjour，如 `_ultrasend._tcp`）。 |
| **屏幕录制 / 辅助捕获类能力** | 用于托盘中的**截图与标注**等功能（与系统 `NSScreenCaptureUsageDescription` 一致）。 |

### 2.5 Windows / Linux（桌面）

通常依赖网络与本地文件系统访问；具体能力以各发行版安装包及系统提示为准。

### 2.6 HarmonyOS（鸿蒙）

典型声明包括：**互联网**、**获取 Wi‑Fi 信息**、**相机**（用于扫码等），用于网络连接、局域网相关能力与扫码场景。

当您使用 **Web 端**时，我们可能使用：

您可通过浏览器设置管理 Cookie 与站点数据；**清除后可能导致需要重新登录或重新选择偏好**。

## 7. 存储地点与跨境传输

**本政策（中国大陆版）下的个人信息，原则上存储在中华人民共和国境内处理。** 如因使用境外第三方（例如部分云服务或支付回执链路）涉及跨境传输，我们将依法履行安全评估、标准合同、认证或单独同意等义务（如适用），并在具体场景向您说明。
