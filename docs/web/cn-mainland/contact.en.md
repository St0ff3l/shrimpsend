## What to include

To help us diagnose issues faster, include:

1. Platform: iOS, Android, Windows, macOS, Web, or HarmonyOS.
2. Network: same LAN, cross-network, corporate network, or mobile network.
3. Whether you use custom S3.
4. Error time and approximate reproduction steps.
5. For file transfer issues, file size and transfer mode.
